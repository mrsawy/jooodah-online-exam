## Joodah Sales Quiz


## Installation

```bash
# Clone the repository
git clone https://github.com/mrsawy/jooodah-online-exam.git

# Change into the project directory
cd your-repository

# Change into the backend
cd your-repository/backend
# Install dependencies
npm install
# run the backend
node index
```

```bash
#in another terminal :

# Change into the frontend folder of the project
cd your-repository/frontend

# Install dependencies
npm install
# run the backend
npm start
```
